# DDClient addon for hass.io
[![Build Status](https://travis-ci.org/ritschard/hassio-addons.svg?branch=master)](https://travis-ci.org/ritschard/hassio-addons)[![](https://images.microbadger.com/badges/version/ritschard/armhf-ddclient.svg)](https://microbadger.com/images/ritschard/armhf-ddclient "Get your own version badge on microbadger.com")

## Configuration

### daemon (int)
Default: 60
Check every x seconds.

### syslog (str)
Default: yes
Log update msgs to syslog.

### login (str)
Your username.

### password (str)
Your password or MD5/SHA256 of password.

### server (str)
Default: api.dynu.com
IP update server.

### protocol (str)
Default: dyndns2
Log update msgs to syslog.

### ssl (str)
Default: yes
Log update msgs to syslog.
